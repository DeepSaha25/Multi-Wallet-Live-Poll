# Screenshots

## wallet-options.png

Screenshot showing the wallet connection modal with available wallet options (Freighter, Albedo, Physics Wallet, etc.).

**To add this image:**
1. Run the app: `npm run dev` in the `frontend` directory
2. Open browser to http://localhost:5173
3. Click "Connect Wallet" button
4. Take a screenshot of the wallet selection modal
5. Save as `wallet-options.png` in this directory

This demonstrates the multi-wallet integration feature of the Live Poll app.
